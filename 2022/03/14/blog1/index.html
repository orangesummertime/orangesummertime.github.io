用Hexo Admin写博客

安装Hexo Admin插件

在博客根目录下右键打开Git Bash，键入“npm install –save hexo-admin”（若网络错误，则重试or搭梯）。

进入编辑界面

在Git Bash界面，键入
“hexo server -d”

随后打开http://localhost:4000/admin/ ，即进入Hexo Admin图形界面。

看板娘

安装依赖

Git Bash -> npm install –save hexo-helper-live2d
若出现 “run ‘npm audit fix”……的错误”，键入npm audit fix force
若成功，则在node_modules目录下出现live2d-widget目录

安装看板娘

可用npm安装模型文件，e.g.:
npm install live2d-widget-model-shizuku
model名字可在live2d-widget-models中找到

看板娘登场

在主题或全局的_config.yml中添加如下语句

## Live2D看板娘
live2d:
  enable: true
  pluginModelPath: assets/
  model:
    # 模板目录，在node_modules里
    use: live2d-widget-model-shizuku  
  display:
    position: right
    width: 300 
    height: 600
  mobile:
    # 在手机端显示
    show: false   
  rect:
    opacity:0.7

插入音乐

安装依赖

Git Bash -> npm install –save hexo-tag-aplayer

启用Meting播放器

在全局_config.yml中添加以下语句：
aplayer:
meting: true

实际应用

在网页md文件中加入如下语句

{% meting "3986040" "netease" "song" "theme:#555" "mutex:true" "listmaxheight:340px" "preload:auto" %}
