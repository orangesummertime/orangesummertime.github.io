用Hexo Admin写博客

安装Hexo Admin插件

在博客根目录下右键打开Git Bash，键入“npm install –save hexo-admin”（若网络错误，则重试or搭梯）。

进入编辑界面

在Git Bash界面，键入
“hexo server -d”

随后打开http://localhost:4000/admin/ ，即进入Hexo Admin图形界面。

安装依赖

Git Bash -> npm install –save hexo-helper-live2d
若出现 “run ‘npm audit fix”……的错误”，键入npm audit fix force
若成功，则在node_modules目录下出现live2d-widget目录

安装看板娘

可用npm安装模型文件，e.g.:
npm install live2d-widget-model-shizuku
